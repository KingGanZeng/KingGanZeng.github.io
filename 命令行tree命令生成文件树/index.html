命令行 效果
tree -d 只显示文件夹
tree -D 显示文件的最后修改时间
tree -L n n表示显示项目的层级，n=3即只显示项目的三层结构
tree -I pattern pattern表示想要过滤的目录，例如 tree -I “node_modules”可以过滤掉node_modules这个文件夹

注意，这里的参数-d、-D区分大小写，大家可以自行实验。
